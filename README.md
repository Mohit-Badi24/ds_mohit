# Trader Behavior vs Market Sentiment Analysis

## Overview
This project analyzes how trader behavior and performance change under different Bitcoin market sentiment regimes using the Fear & Greed Index and historical trade-level data.

## Project Structure
- notebook_1.ipynb: Complete data analysis and modeling workflow
- csv_files/: Cleaned and processed datasets
- outputs/: Visualizations generated during analysis
- ds_report.pdf: Final summarized insights and conclusions

## Methodology
- Data cleaning and preprocessing
- Feature engineering
- Daily aggregation of trader metrics
- Alignment with market sentiment
- Exploratory and pattern-based analysis

- Python
- Pandas
- Matplotlib
